## 📖 About The Project

**PharmaLogic** is a smart drug interaction analysis system that helps healthcare professionals and patients detect potential drug interactions instantly. The system provides risk assessment, lab monitoring alerts, and personalized recommendations based on patient profiles.

### 🎯 Key Features

- ✅ **Real-time Drug Interaction Detection** - Analyze interactions between multiple medications
- 📊 **Risk Assessment** - Clear risk levels (High/Moderate/Low/None)
- 🧪 **Lab Monitoring Alerts** - Automated lab test recommendations
- 👤 **User Authentication** - JWT-based secure authentication
- 👨‍⚕️ **Role-based Access** - Patient, Doctor, and Pharmacist roles
- 📄 **PDF Export** - Generate professional interaction reports
- 🎨 **Modern UI** - Responsive design for all devices

## 🏗️ Built With

- **Backend**: FastAPI, SQLAlchemy, SQLite
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Authentication**: JWT (JSON Web Tokens)
- **Security**: bcrypt for password hashing
